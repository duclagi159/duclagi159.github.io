TikTok connection troubleshooting

  1. Open Tool Beautiful and select Manager Account.
  2. Confirm the TikTok application configuration and redirect URI are correct.
  3. Select Connect TikTok and complete authorization on TikTok's official page.
  4. After authorization, return to Tool Beautiful and synchronize the account.
  5. If a connection is no longer needed, select Disconnect and revoke access in TikTok settings.
