Data Deletion Instructions

Effective and last updated: July 16, 2026

Tool Beautiful stores TikTok connection information and account snapshots locally on the user's Windows computer. Follow these steps to stop future access and remove local data.

1. Disconnect inside Tool Beautiful

  1. Open Tool Beautiful.
  2. Open TikTok Beta → Manager Account.
  3. Select the connected account.
  4. Select Disconnect and confirm the action.

This sends an access-revocation request when available and removes the account connection from the local Manager Account store.

2. Revoke access in TikTok

Sign in to TikTok, open the security or connected-app settings, locate Tool Beautiful, and remove its access. TikTok's menu wording may vary by region and app version.

3. Remove remaining local application data

After closing Tool Beautiful, users who need to remove the complete Manager Account configuration may delete this local file:

%USERPROFILE%\.toolbeautiful\manager_account_config.json

The file contains local settings and encrypted authorization information. Deleting it removes the locally stored Manager Account configuration. It does not delete content or information independently retained by TikTok.

4. Request assistance

If the application is unavailable or assistance is required, submit a request through the Support page. Do not post access tokens, refresh tokens, client secrets, passwords, or cookies.

What deletion does not do

  • It does not delete the user's TikTok account.
  • It does not delete videos already published on TikTok.
  • It does not control data independently retained by TikTok under TikTok's policies.
